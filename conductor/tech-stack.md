# Tech Stack

## Backend
- **Language:** Python 3.9+
- **Framework:** FastAPI
- **Web Server:** Uvicorn
- **Environment Management:** python-dotenv
- **File Uploads:** python-multipart
- **Utilities:** Slowapi (Rate limiting), Requests

## Frontend
- **Framework:** React 18+
- **Build Tool:** Vite
- **Language:** TypeScript
- **Routing:** React Router 7
- **State Management:** Zustand
- **Server State & Data Fetching:** TanStack React Query
- **UI Icons:** Lucide React
- **Animations:** Framer Motion
- **Notifications:** Sonner

## Services & Infrastructure
- **Database:** Firebase Firestore (managed via `firebase-admin` and `google-cloud-firestore`)
- **Speech-to-Text (STT):** Deepgram
- **AI/LLM Service:** Vertex AI (Google Cloud Platform)
- **PDF Generation:** Custom implementation (Python)

## Testing
- **E2E Testing:** Playwright (TypeScript)
